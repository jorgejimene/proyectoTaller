Calcula la aceleración gravitatoria debida al campo armónico de un cuerpo central.

Esta función utiliza coeficientes armónicos (Cnm, Snm) hasta un grado y orden máximos para calcular la aceleración gravitatoria en un sistema inercial.

Parameters
rVector de posición del satélite en el sistema inercial (3x1).
EMatriz de transformación del sistema inercial al sistema fijo del cuerpo (3x3).
n_maxGrado máximo del desarrollo armónico.
m_maxOrden máximo del desarrollo armónico.
Returns
Aceleración del satélite (3x1) en el sistema inercial.
