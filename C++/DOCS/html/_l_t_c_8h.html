Detailed Description

Transformación del sistema del meridiano de Greenwich a coordenadas tangentes locales.

Esta función genera una matriz de rotación que transforma coordenadas geodésicas (longitud y latitud) al sistema de coordenadas tangente local (Este-Norte-Zenit, ENZ).

Basado en la implementación original en MATLAB por M. Mahooti (2015/08/12).
